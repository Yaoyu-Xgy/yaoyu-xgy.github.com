构建过程中干了不少智障事儿。

尝试用不同分支托管 hexo 站点 和 生成站点 根目录

Github Action 的触发条件是push到main分支；而执行脚本包括hexo deploy；后者会再次git push……如此循环。

托管 hexo 站点目录到新仓库
