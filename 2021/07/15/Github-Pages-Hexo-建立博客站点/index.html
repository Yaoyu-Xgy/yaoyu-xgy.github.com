过程

配置 Github 帐号

  1. 新建仓库：建议Repository Name设为userName.github.io ，因为此时该仓库会被视作该Github帐号的个人主页而获得userName.github.io的域名；而其他仓库的默认*.github.io地址为username.github.io/repositoryName。

    若选择了非个人主页仓库建立博客页面，需要更改_config.yml中的url

  2. 可选：新建并导入一个 SSH 密钥对，可参考 github: Generating a new SSH key and adding it to the ssh-agent。
